# CertifyIQ — Quiz Web App

## Quick Start

```bash
# 1. Install dependencies (one time)
npm install express cors

# 2. Start the server
node server.js
```

App runs at: http://localhost:3000
Webhook URL: http://localhost:3000/webhook/quiz

## Files
- `index.html` — The quiz web app (served automatically)
- `server.js`  — Express server (webhook receiver + static file server)
